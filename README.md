# next-full-stack-demo

## 项目概述

这是一个基于 Next.js App Router 构建的全栈项目，包含以下技术栈：
- 前端框架：Next.js 14
- 样式引擎：Tailwind CSS
- 状态管理：tRPC
- 数据库 ORM：Drizzle
- 身份验证：NextAuth.js
- 代码质量：ESLint + TypeScript

主要功能模块：
- 基于文件路由的 API 端点（app/api）
- 类型安全的 tRPC 通信（server/api/trpc.ts）
- 全局字体优化（Geist 字体集成）
- 响应式布局系统（app/layout.tsx）
